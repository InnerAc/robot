#ExpertRobot
中国软件杯赛题

##项目介绍
基于大数据处理技术构建“比特能·专家机器人”，延伸专家的大脑，通过网络为用户提供查询、搜索、提问、推送与咨询等各类服务。 

##开发环境
* ubuntu 系统或 windows 下的 cygwin 环境
* Hadoop 2.4.0 及以上
* Hbase 0.98.6.1 及以上（hbase版本需要和hadoop相对应）
* JDK 1.7 及以上（openjdk-7-jdk 或者 jdk-7u71）
* Eclipse Luna (调试 Hadoop 推荐IDE)
* Jena 2.13.0
* Maven
